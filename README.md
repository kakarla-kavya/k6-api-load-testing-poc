# API Load Testing Using K6
### Prerequisites

1. Clone repository:
   ``` bash 
   git clone git@github.com:kakarla-kavya/k6-api-load-testing-poc.git  
   ```
2. Install the K6 using below command  
    ```bash 
    brew install k6 
    ``` 
3. Install node 
    ``` bash
    brew install node
    ```

 ### Running test
 1. Run test
  ``` bash  
   npm test <<filename>>
   ```
